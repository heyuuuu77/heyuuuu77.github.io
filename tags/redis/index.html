数据类型 String: 字符串 Hash: 哈希, 是一个键值对集合，类似于 Python 中的字典。适合存储对象，一个哈希可以包含多个字段和对应的值，每个哈希最多可存储2^32 -1个键值对。 常见命令: 设置单个字段值：HSET key field value，例如 HSET user:1 name “Bob”。 获取单个字段值：HGET key field，例如 HGET user:1 name 会返回 “Bob”。 获取所有字段和值：HGETALL key。 List: 列表 Set: 集合 Sorted Set: 有序集合
